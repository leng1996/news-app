本项目是用nodejs搭建一个接口，配合mongodb数据库和vue等为主要技术栈
完成的一个类似新闻客户端的移动段app。

jiekou.txt是后端node接口文档，
mongodb.txt记载了常见的mongodb的操作方式，
mongo文件夹是数据入库的留存文件，
node_modules可不下载，npm i即可，
public文件包含已经打包好的vue所有项目文件。

本地环境下访问：http://localhost:3000
本项目线上ip为：http://120.27.250.91：3000（域名备案还未通过）
